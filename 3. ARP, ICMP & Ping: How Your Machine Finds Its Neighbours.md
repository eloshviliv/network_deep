# рЃЊрЃдрЃћ 3 Рђћ ARP, ICMP рЃЊрЃљ Ping: рЃарЃЮрЃњрЃЮрЃа рЃърЃЮрЃБрЃџрЃЮрЃЉрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљ рЃЌрЃљрЃЋрЃўрЃА рЃЏрЃћрЃќрЃЮрЃЉрЃџрЃћрЃЉрЃА

## рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃћрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў

рЃДрЃЋрЃћрЃџрЃљ рЃЦрЃАрЃћрЃџрЃБрЃарЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ Рђћ рЃарЃљрЃф рЃљрЃа рЃБрЃюрЃЊрЃљ рЃарЃЌрЃБрЃџрЃў рЃўрЃДрЃЮрЃА Рђћ рЃўрЃгрЃДрЃћрЃЉрЃљ рЃћрЃарЃЌрЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃЌ:
**"рЃЋрЃўрЃА рЃљрЃЦрЃЋрЃА рЃћрЃА IP?"**

рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃАрЃћрЃарЃЋрЃћрЃарЃў ping-рЃљрЃЋрЃА gateway-рЃА, рЃЌрЃЦрЃЋрЃћрЃюрЃў pod рЃБрЃФрЃљрЃ«рЃћрЃЉрЃА cluster service-рЃА, рЃЌрЃБ рЃЌрЃЦрЃЋрЃћрЃюрЃў VM рЃљрЃдрЃгрЃћрЃЋрЃА рЃњрЃљрЃарЃћ API-рЃАРђд рЃЌрЃЦрЃЋрЃћрЃюрЃЏрЃљ рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљрЃЏ рЃБрЃюрЃЊрЃљ рЃўрЃфрЃЮрЃЊрЃћрЃА рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃўрЃА MAC рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў (рЃљрЃю рЃЏрЃўрЃАрЃў gateway) рЃАрЃљрЃюрЃљрЃЏ рЃћрЃарЃЌрЃў рЃърЃљрЃЎрЃћрЃбрЃўрЃф рЃЎрЃў рЃЊрЃљрЃбрЃЮрЃЋрЃћрЃЉрЃА рЃЏрЃљрЃА.

рЃЊрЃдрЃћрЃА рЃЕрЃЋрЃћрЃю рЃљрЃЋрЃ«рЃАрЃюрЃўрЃЌ рЃљрЃЏ рЃљрЃдрЃЏрЃЮрЃЕрЃћрЃюрЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃўрЃА рЃАрЃљрЃЏ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊ рЃюрЃљрЃгрЃўрЃџрЃА:

* **ARP** Рєњ рЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃА IP Рєњ MAC
* **ICMP** Рєњ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃбрЃЮрЃЎрЃЮрЃџрЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃљрЃдрЃЏрЃЮрЃЕрЃћрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА
* **Ping** Рєњ ICMP Echo рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃерЃћрЃАрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃљрЃЊ

рЃћрЃА рЃљрЃарЃўрЃА рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃў DevOps & SRE troubleshooting-рЃўрЃАрЃЌрЃЋрЃўрЃА.

## рЃюрЃљрЃгрЃўрЃџрЃў 1 Рђћ ARP: Address Resolution Protocol

ARP рЃърЃљрЃАрЃБрЃ«рЃЮрЃЉрЃА рЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃќрЃћ:
**"рЃарЃЮрЃЏрЃћрЃџ MAC рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃА рЃљрЃЦрЃЋрЃА рЃћрЃА IP?"**

рЃЌрЃБ рЃњрЃАрЃБрЃарЃЌ рЃърЃљрЃЎрЃћрЃбрЃўрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ `192.168.1.20`-рЃќрЃћ, рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљ рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА ARP-рЃА.

### рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ

рЃЌрЃЦрЃЋрЃћрЃю рЃњрЃљрЃЦрЃЋрЃЌ рЃЋрЃўрЃдрЃљрЃфрЃўрЃА рЃЦрЃБрЃЕрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў (IP), рЃЏрЃљрЃњрЃарЃљрЃЏ рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ рЃЏрЃљрЃЌрЃў рЃАрЃљрЃ«рЃћ (MAC) рЃарЃљрЃЌрЃљ рЃгрЃћрЃарЃўрЃџрЃў рЃърЃўрЃарЃљрЃЊрЃљрЃЊ рЃњрЃљрЃЊрЃљрЃАрЃфрЃћрЃЌ.

### рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА:

**1. рЃЌрЃЦрЃЋрЃћрЃю рЃћрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃўрЃЌ рЃЏрЃЌрЃћрЃџ рЃЦрЃАрЃћрЃџрЃА:**
```
рЃЋрЃўрЃА рЃљрЃЦрЃЋрЃА 192.168.1.20? рЃБрЃЌрЃ«рЃљрЃарЃўрЃЌ 192.168.1.10-рЃА
```

**2. рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљ рЃљрЃЏ IP-рЃЌрЃў рЃърЃљрЃАрЃБрЃ«рЃЮрЃЉрЃА:**
```
192.168.1.20 рЃљрЃарЃўрЃА 02:42:ac:11:00:03-рЃќрЃћ
```

**3. рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљ рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃљрЃЏрЃљрЃА ARP cache-рЃерЃў.**

### ARP cache-рЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ:

```bash
ip neigh
```

## рЃарЃћрЃљрЃџрЃБрЃарЃў DevOps рЃАрЃўрЃбрЃБрЃљрЃфрЃўрЃћрЃЉрЃў рЃАрЃљрЃЊрЃљрЃф ARP рЃўрЃерЃџрЃћрЃЉрЃљ

* РЮї **Pod/node рЃљрЃа рЃерЃћрЃБрЃФрЃџрЃўрЃљ gateway-рЃА рЃЏрЃўрЃдрЃгрЃћрЃЋрЃљ** Рєњ ARP incomplete
* РЮї **рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃљрЃЊ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў VLAN-рЃћрЃЉрЃў** Рєњ ARP рЃўрЃерЃџрЃћрЃЉрЃљ
* РЮї **рЃЊрЃБрЃЉрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў IP-рЃћрЃЉрЃў** Рєњ ARP рЃфрЃЋрЃџрЃўрЃА рЃЮрЃа MAC-рЃА рЃерЃЮрЃарЃўрЃА
* РЮї **рЃ«рЃЏрЃћрЃџрЃў ARP рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃћрЃЉрЃў** Рєњ рЃўрЃюрЃбрЃћрЃарЃЏрЃўрЃбрЃћрЃюрЃбрЃБрЃџрЃў рЃЎрЃљрЃЋрЃерЃўрЃарЃў
* РЮї **ARP poisoning** Рєњ рЃЌрЃљрЃЋрЃЊрЃљрЃЏрЃАрЃ«рЃЏрЃћрЃџрЃў рЃ«рЃћрЃџрЃА рЃБрЃерЃџрЃўрЃА рЃбрЃарЃљрЃцрЃўрЃЎрЃА

### ARP рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

```
192.168.1.1 dev eth0  INCOMPLETE
```

рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃЌрЃЦрЃЋрЃћрЃюрЃЏрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃЏ рЃўрЃЎрЃўрЃЌрЃ«рЃљ "рЃЋрЃўрЃА рЃљрЃЦрЃЋрЃА 192.168.1.1?" рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃарЃљрЃЋрЃўрЃю рЃБрЃърЃљрЃАрЃБрЃ«рЃљ.

## рЃюрЃљрЃгрЃўрЃџрЃў 2 Рђћ ICMP: Internet Control Message Protocol

ICMP рЃљрЃа рЃљрЃарЃўрЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃбрЃЮрЃЎрЃЮрЃџрЃў. рЃћрЃА рЃљрЃарЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃўрЃАрЃљ рЃЊрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃљрЃюрЃњрЃљрЃарЃўрЃерЃњрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃбрЃЮрЃЎрЃЮрЃџрЃў.

### рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ:

* **Ping** (echo request/reply)
* **TTL рЃњрЃљрЃЊрЃљрЃГрЃљрЃарЃЉрЃћрЃЉрЃБрЃџрЃў** (traceroute)
* **рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљ рЃЏрЃўрЃБрЃгрЃЋрЃЊрЃЮрЃЏрЃћрЃџрЃў**
* **рЃњрЃљрЃЊрЃљрЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў**
* **рЃърЃљрЃЎрЃћрЃбрЃў рЃФрЃљрЃџрЃўрЃљрЃю рЃЊрЃўрЃЊрЃўрЃљ** (MTU рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃў)

### рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ

ICMP рЃљрЃарЃўрЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃљрЃцрЃЮрЃАрЃбрЃЮ рЃЌрЃљрЃюрЃљрЃЏрЃерЃарЃЮрЃЏрЃћрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃфрЃюрЃЮрЃЉрЃћрЃЉрЃЌ:
* "рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃљрЃЏрЃљрЃюрЃљрЃЌрЃў рЃЋрЃћрЃа рЃЏрЃўрЃгрЃЮрЃЊрЃћрЃЉрЃБрЃџрЃљ."
* "рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА."
* "рЃњрЃќрЃљ рЃФрЃљрЃџрЃўрЃљрЃю рЃњрЃарЃФрЃћрЃџрЃўрЃљ."

## рЃюрЃљрЃгрЃўрЃџрЃў 3 Рђћ Ping: рЃБрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃАрЃў рЃЦрЃАрЃћрЃџрЃБрЃарЃў рЃбрЃћрЃАрЃбрЃў (ICMP Echo)

Ping рЃўрЃДрЃћрЃюрЃћрЃЉрЃА ICMP Echo (рЃбрЃўрЃърЃў 8) рЃЊрЃљ Echo Reply (рЃбрЃўрЃърЃў 0).

рЃарЃЮрЃфрЃљ рЃњрЃљрЃБрЃерЃЋрЃћрЃЉрЃЌ:
```bash
ping 8.8.8.8
```

### рЃ«рЃЊрЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў:

1. DNS рЃарЃћрЃќрЃЮрЃџрЃБрЃфрЃўрЃљ (рЃЮрЃцрЃфрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў)
2. ARP рЃарЃћрЃќрЃЮрЃџрЃБрЃфрЃўрЃљ (рЃЌрЃБ рЃўрЃЏрЃљрЃЋрЃћ subnet-рЃерЃўрЃљ)
3. ICMP Echo Request рЃњрЃљрЃўрЃњрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ
4. ICMP Echo Reply рЃЏрЃўрЃўрЃдрЃћрЃЉрЃљ

### Ping рЃњрЃћрЃБрЃЉрЃюрЃћрЃЉрЃљрЃЌ:

* РюЁ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ рЃ░рЃЮрЃАрЃбрЃў?
* РЈ▒№ИЈ рЃарЃљ рЃљрЃарЃўрЃА latency?
* ­ЪЊЅ рЃ«рЃЊрЃћрЃЉрЃљ packet loss?
* ­ЪЏБ№ИЈ рЃАрЃбрЃљрЃЉрЃўрЃџрЃБрЃарЃўрЃљ рЃЏрЃљрЃарЃерЃарЃБрЃбрЃў?

## Troubleshooting рЃарЃЮрЃњрЃЮрЃарЃф SRE

### 1. рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ 1: "Ping рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЏрЃљрЃњрЃарЃљрЃЏ curl рЃљрЃарЃљ."

**рЃюрЃўрЃерЃюрЃљрЃЋрЃА:**
* РюЁ L3 рЃарЃўрЃњрЃќрЃћрЃљ
* РЮї DNS рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃўрЃДрЃЮрЃА
* РЮї Firewall рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЉрЃџрЃЮрЃЎрЃљрЃЋрЃА рЃърЃЮрЃарЃбрЃА 80/443
* РЮї рЃљрЃърЃў рЃарЃћрЃЏрЃЮрЃБрЃЌрЃќрЃћ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃарЃЌрЃБрЃџрЃў рЃўрЃДрЃЮрЃА

## рЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў (рЃЊрЃдрЃћ 4)

рЃ«рЃЋрЃљрЃџ рЃБрЃцрЃарЃЮ рЃдрЃарЃЏрЃљрЃЊ рЃерЃћрЃЋрЃљрЃџрЃЌ:
**рЃЊрЃдрЃћ 4 Рђћ IP Routing рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў: рЃарЃЮрЃњрЃЮрЃа рЃЏрЃЮрЃФрЃарЃљрЃЮрЃЉрЃћрЃю рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў рЃЦрЃАрЃћрЃџрЃћрЃЉрЃерЃў**

рЃЌрЃЦрЃЋрЃћрЃю рЃњрЃљрЃўрЃњрЃћрЃЉрЃЌ:

* рЃарЃљ рЃљрЃарЃўрЃА routing table
* рЃарЃЮрЃњрЃЮрЃа рЃўрЃарЃЕрЃћрЃЋрЃА Linux рЃерЃћрЃЏрЃЊрЃћрЃњ hop-рЃА
* рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА default gateway
* рЃарЃљрЃА рЃарЃћрЃљрЃџрЃБрЃарЃљрЃЊ рЃюрЃўрЃерЃюрЃљрЃЋрЃА CIDR routes-рЃўрЃАрЃЌрЃЋрЃўрЃА
* рЃарЃљрЃбрЃЮрЃЏ рЃўрЃдрЃћрЃЉрЃћрЃю рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў рЃќрЃЮрЃњрЃ»рЃћрЃа рЃБрЃфрЃюрЃљрЃБрЃа рЃњрЃќрЃћрЃЉрЃА
* рЃарЃЮрЃњрЃЮрЃа рЃЊрЃљрЃЋрЃљрЃЊрЃћрЃЉрЃљрЃњрЃЮрЃЌ routing рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃў `ip route`, `mtr` рЃЊрЃљ `traceroute` рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ

рЃћрЃА рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃърЃЮрЃАрЃбрЃў рЃЏрЃЌрЃћрЃџ рЃАрЃћрЃарЃўрЃљрЃерЃў.
