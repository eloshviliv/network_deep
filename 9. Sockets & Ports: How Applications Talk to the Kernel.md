# рЃЊрЃдрЃћ 9 Рђћ рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃў рЃЊрЃљ рЃърЃЮрЃарЃбрЃћрЃЉрЃў: рЃарЃЮрЃњрЃЮрЃа рЃАрЃљрЃБрЃЉрЃарЃЮрЃЉрЃћрЃю рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃў Kernel-рЃЌрЃљрЃю

рЃњрЃарЃФрЃћрЃџрЃў, рЃФрЃљрЃџрЃўрЃљрЃю рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃЌрЃљрЃЌрЃЋрЃўрЃА рЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃдрЃарЃЏрЃљрЃЊ рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃљрЃ«рЃАрЃюрЃљ рЃЦрЃАрЃћрЃџрЃћрЃЉрЃўрЃА рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃЎрЃЮрЃюрЃфрЃћрЃцрЃфрЃўрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ.

## рЃарЃљрЃбрЃЮрЃЏ рЃљрЃЦрЃЋрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃА (рЃЏрЃћрЃбрЃў рЃЋрЃўрЃЊрЃарЃћ рЃарЃћрЃљрЃџрЃБрЃарЃљрЃЊ рЃцрЃўрЃЦрЃарЃЮрЃЉрЃЌ)

рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃцрЃўрЃЦрЃарЃЮрЃЌ, рЃарЃЮрЃЏ рЃърЃЮрЃарЃбрЃћрЃЉрЃў рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ 80, 22, рЃљрЃю 443.
рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃцрЃўрЃЦрЃарЃЮрЃЌ, рЃарЃЮрЃЏ рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃў рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃарЃљрЃдрЃљрЃфрЃљрЃљ, рЃарЃљрЃАрЃљрЃф рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃћрЃЉрЃў рЃЦрЃЏрЃюрЃўрЃљрЃю.

рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃўрЃЏрЃљрЃарЃЌрЃџрЃћ рЃўрЃАрЃљрЃљ:

**рЃДрЃЋрЃћрЃџрЃљ рЃЦрЃАрЃћрЃџрЃўрЃА рЃЎрЃљрЃЋрЃерЃўрЃарЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљ рЃЦрЃЏрЃюрЃўрЃА, рЃљрЃарЃўрЃА рЃАрЃЮрЃЎрЃћрЃбрЃў.**
**рЃДрЃЋрЃћрЃџрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃљрЃЏрЃЪрЃдрЃљрЃЋрЃюрЃћрЃЉрЃЌ, рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃърЃЮрЃарЃбрЃћрЃЉрЃА.**
**рЃДрЃЋрЃћрЃџрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃљрЃБрЃЉрЃарЃЮрЃЉрЃА рЃЦрЃАрЃћрЃџрЃўрЃЌ, рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃА.**

рЃарЃЮрЃфрЃљ рЃЌрЃЦрЃЋрЃћрЃю:
- рЃ«рЃАрЃюрЃўрЃЌ рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃА
- рЃљрЃерЃЋрЃћрЃЉрЃЌ рЃЏрЃўрЃЎрЃарЃЮрЃАрЃћрЃарЃЋрЃўрЃАрЃА
- ssh-рЃўрЃЌ рЃерЃћрЃЊрЃўрЃ«рЃљрЃарЃЌ рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ
- curl-рЃўрЃЌ рЃБрЃФрЃљрЃ«рЃћрЃЉрЃЌ API-рЃА
- рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃўрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃА
- рЃљрЃерЃџрЃўрЃЌ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃА Kubernetes-рЃерЃў
- рЃљрЃЏрЃЪрЃдрЃљрЃЋрЃюрЃћрЃЉрЃЌ рЃърЃЮрЃарЃбрЃћрЃЉрЃА Docker-рЃерЃў
- рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃбрЃЋрЃўрЃарЃЌрЃўрЃА рЃњрЃљрЃЏрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃџрЃћрЃЉрЃА

... рЃЌрЃЦрЃЋрЃћрЃю рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃА + рЃърЃЮрЃарЃбрЃћрЃЉрЃА.

рЃЊрЃдрЃћрЃА, рЃљрЃЋрЃ«рЃАрЃюрЃў рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА:
- рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃЌ
- рЃЊрЃўрЃљрЃњрЃарЃљрЃЏрЃћрЃЉрЃўрЃЌ
- рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљрЃерЃџрЃћрЃЉрЃўрЃЌ
- рЃарЃћрЃљрЃџрЃБрЃарЃў DevOps/SRE рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃўрЃЌ
- рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃЌ (`ss`, `lsof`, `netstat`)

рЃћрЃА рЃљрЃарЃўрЃА рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃЌрЃљрЃЋрЃў рЃЦрЃАрЃћрЃџрЃћрЃЉрЃерЃў.

## рЃюрЃљрЃгрЃўрЃџрЃў 1 Рђћ рЃарЃљ рЃљрЃарЃўрЃА рЃърЃЮрЃарЃбрЃў? (рЃљрЃ«рЃАрЃюрЃўрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф 10 рЃгрЃџрЃўрЃА рЃЉрЃљрЃЋрЃерЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА)

рЃгрЃљрЃарЃЏрЃЮрЃўрЃЊрЃњрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў рЃљрЃарЃўрЃА рЃЊрЃўрЃЊрЃў рЃАрЃљрЃфрЃ«рЃЮрЃЋрЃарЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃюрЃЮрЃЉрЃљ.
рЃерЃћрЃюрЃЮрЃЉрЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ рЃфрЃ«рЃЮрЃЋрЃарЃЮрЃЉрЃА рЃЉрЃћрЃЋрЃарЃў рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃў (рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃў).

**рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃерЃћрЃюрЃЮрЃЉрЃљрЃА рЃљрЃЦрЃЋрЃА 65,536 рЃЎрЃљрЃарЃў (рЃърЃЮрЃарЃбрЃћрЃЉрЃў).**

рЃДрЃЋрЃћрЃџрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃфрЃ«рЃЮрЃЋрЃарЃЮрЃЉрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЎрЃљрЃарЃўрЃА рЃБрЃЎрЃљрЃю.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:
- **рЃърЃЮрЃарЃбрЃў 22** Рєњ SSH
- **рЃърЃЮрЃарЃбрЃў 80** Рєњ HTTP рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў
- **рЃърЃЮрЃарЃбрЃў 443** Рєњ HTTPS рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў
- **рЃърЃЮрЃарЃбрЃў 3306** Рєњ MySQL
- **рЃърЃЮрЃарЃбрЃў 6379** Рєњ Redis
- **рЃърЃЮрЃарЃбрЃў 5432** Рєњ Postgres

рЃЌрЃБ рЃњрЃАрЃБрЃарЃЌ рЃћрЃгрЃЋрЃўрЃЮрЃЌ рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃфрЃ«рЃЮрЃЋрЃарЃЮрЃЉрЃА рЃЉрЃўрЃюрЃљ #22-рЃерЃў Рєњ рЃЏрЃўрЃЊрЃўрЃ«рЃљрЃарЃЌ рЃЎрЃљрЃарЃў 22-рЃќрЃћ.

рЃўрЃњрЃўрЃЋрЃћ рЃњрЃќрЃўрЃЌ:
рЃЌрЃБ рЃЎрЃџрЃўрЃћрЃюрЃбрЃА рЃАрЃБрЃарЃА рЃћрЃАрЃљрЃБрЃЉрЃарЃЮрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃА SSH рЃАрЃћрЃарЃЋрЃћрЃарЃА Рєњ рЃўрЃА рЃЏрЃўрЃЊрЃўрЃА рЃърЃЮрЃарЃб 22-рЃќрЃћ.

**рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃгрЃћрЃАрЃў:**
IP рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃА рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљрЃА, рЃърЃЮрЃарЃбрЃў рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃА.

## рЃюрЃљрЃгрЃўрЃџрЃў 2 Рђћ рЃарЃљ рЃљрЃарЃўрЃА рЃАрЃЮрЃЎрЃћрЃбрЃў? (рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃљрЃ«рЃАрЃюрЃўрЃџрЃў)

рЃАрЃЮрЃЎрЃћрЃбрЃў рЃљрЃарЃўрЃА:

**рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃљ:**
IP + рЃърЃЮрЃарЃбрЃў + рЃърЃарЃЮрЃбрЃЮрЃЎрЃЮрЃџрЃў (TCP/UDP)

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:
```
192.168.1.10:8080
```
рЃћрЃА рЃљрЃарЃўрЃА рЃАрЃЮрЃЎрЃћрЃбрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў.

рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃарЃўрЃА рЃЎрЃўрЃЊрЃћрЃЋ рЃћрЃарЃЌрЃў рЃюрЃљрЃЉрЃўрЃ»рЃў:
рЃарЃЮрЃфрЃљ рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃЊрЃљрЃЏрЃДрЃљрЃарЃЊрЃћрЃЉрЃљ, Linux рЃўрЃДрЃћрЃюрЃћрЃЉрЃА 4-tuple-рЃА:

```
рЃгрЃДрЃљрЃарЃЮрЃА IP
рЃгрЃДрЃљрЃарЃЮрЃА рЃърЃЮрЃарЃбрЃў
рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃўрЃА IP
рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃўрЃА рЃърЃЮрЃарЃбрЃў
```

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:
```
рЃЎрЃџрЃўрЃћрЃюрЃбрЃў:  10.0.0.5:55012
рЃАрЃћрЃарЃЋрЃћрЃарЃў:  142.250.183.14:443
```

рЃћрЃарЃЌрЃљрЃЊ:
```
10.0.0.5:55012 Рєњ 142.250.183.14:443
```

рЃћрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃљ рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЎрЃљрЃЋрЃерЃўрЃарЃА.

### рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ

рЃгрЃљрЃарЃЏрЃЮрЃўрЃЊрЃњрЃўрЃюрЃћрЃЌ:
- **рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃАрЃљрЃ«рЃџрЃў = IP рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў**
- **рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЮрЃЌрЃљрЃ«рЃў рЃАрЃљрЃ«рЃџрЃерЃў = рЃърЃЮрЃарЃбрЃў**
- **рЃАрЃљрЃБрЃЉрЃљрЃарЃў рЃЮрЃа рЃЮрЃЌрЃљрЃ«рЃА рЃерЃЮрЃарЃўрЃА рЃЮрЃа рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃАрЃљрЃ«рЃџрЃерЃў = рЃАрЃЮрЃЎрЃћрЃбрЃў**

рЃАрЃЮрЃЎрЃћрЃбрЃў рЃљрЃарЃўрЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃЌрЃЦрЃЏрЃљ:

"рЃЮрЃЌрЃљрЃ«рЃў 55012 рЃАрЃљрЃ«рЃџрЃерЃў 10.0.0.5 рЃАрЃљрЃБрЃЉрЃарЃЮрЃЉрЃА рЃЮрЃЌрЃљрЃ« 443-рЃЌрЃљрЃю рЃАрЃљрЃ«рЃџрЃерЃў Google."

## рЃюрЃљрЃгрЃўрЃџрЃў 3 Рђћ рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃў vs рЃЎрЃџрЃўрЃћрЃюрЃбрЃћрЃЉрЃў

### рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃў рЃБрЃАрЃЏрЃћрЃюрЃћрЃю рЃърЃЮрЃарЃбрЃћрЃЉрЃќрЃћ

рЃўрЃАрЃўрЃюрЃў рЃћрЃџрЃЮрЃЊрЃћрЃЉрЃўрЃљрЃю рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃА.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:
- `sshd` рЃБрЃАрЃЏрЃћрЃюрЃА рЃърЃЮрЃарЃб 22-рЃќрЃћ
- `nginx` рЃБрЃАрЃЏрЃћрЃюрЃА рЃърЃЮрЃарЃб 80/443-рЃќрЃћ
- `postgres` рЃБрЃАрЃЏрЃћрЃюрЃА рЃърЃЮрЃарЃб 5432-рЃќрЃћ

рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ:

```

рЃЊрЃљрЃўрЃюрЃљрЃ«рЃљрЃЋрЃЌ:
```

LISTEN 0 128 0.0.0.0:80   ...
LISTEN 0 128 0.0.0.0:5432 ...

### рЃЎрЃџрЃўрЃћрЃюрЃбрЃћрЃЉрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю ephemeral рЃърЃЮрЃарЃбрЃћрЃЉрЃА

рЃарЃЮрЃфрЃљ рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃўрЃЌ рЃАрЃћрЃарЃЋрЃћрЃарЃА, рЃЌрЃЦрЃЋрЃћрЃюрЃў OS рЃљрЃФрЃџрЃћрЃЋрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ рЃњрЃљрЃЏрЃљрЃЋрЃљрЃџ рЃърЃЮрЃарЃбрЃА:

**рЃЊрЃўрЃљрЃърЃљрЃќрЃЮрЃюрЃў:**
```
49152РђЊ65535 (рЃ«рЃерЃўрЃарЃљрЃЊ)
```

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:
```bash

```

рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА:
```
10.0.0.5:55012 Рєњ 172.217.167.238:443
```
рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃърЃЮрЃарЃбрЃў = 55012 (ephemeral)

## рЃюрЃљрЃгрЃўрЃџрЃў 4 Рђћ рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃърЃЮрЃарЃбрЃћрЃЉрЃў рЃАрЃљрЃГрЃўрЃарЃЮ (рЃБрЃцрЃарЃЮ рЃдрЃарЃЏрЃљ рЃЏрЃўрЃќрЃћрЃќрЃў)

**рЃърЃЮрЃарЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ:**
- рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃА рЃерЃћрЃћрЃФрЃџрЃЮ рЃЦрЃАрЃћрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
- рЃЋрЃћрЃа рЃерЃћрЃФрЃџрЃћрЃЉрЃЊрЃўрЃЌ рЃЉрЃарЃљрЃБрЃќрЃўрЃюрЃњрЃА рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃА рЃЊрЃарЃЮрЃА
- Kubernetes pod-рЃћрЃЉрЃў рЃЋрЃћрЃа рЃАрЃљрЃБрЃЉрЃарЃЮрЃЉрЃЊрЃюрЃћрЃю
- рЃбрЃЋрЃўрЃарЃЌрЃўрЃА рЃњрЃљрЃЏрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃџрЃћрЃЉрЃў рЃЋрЃћрЃа рЃЏрЃўрЃЏрЃљрЃарЃЌрЃљрЃЋрЃЊрЃюрЃћрЃю рЃбрЃарЃљрЃцрЃўрЃЎрЃА
- HTTP, DB, Redis Рђћ рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃ»рЃљрЃ«рЃЊрЃћрЃЉрЃЮрЃЊрЃљ

**рЃърЃЮрЃарЃбрЃћрЃЉрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃЏрЃарЃљрЃЋрЃљрЃџ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃА рЃўрЃќрЃўрЃљрЃарЃЮрЃю рЃўрЃњрЃўрЃЋрЃћ рЃЦрЃАрЃћрЃџрЃўрЃА рЃАрЃбрЃћрЃЎрЃў.**

## рЃюрЃљрЃгрЃўрЃџрЃў 5 Рђћ TCP vs UDP рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃў (рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃљрЃ«рЃАрЃюрЃўрЃџрЃў)

### TCP = рЃбрЃћрЃџрЃћрЃцрЃЮрЃюрЃўрЃА рЃќрЃљрЃарЃў

- рЃЎрЃљрЃЋрЃерЃўрЃарЃќрЃћ рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў
- рЃАрЃљрЃюрЃЊрЃЮ
- рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃБрЃџрЃў
- рЃњрЃљрЃарЃљрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЏрЃўрЃгрЃЮрЃЊрЃћрЃЉрЃљ
- рЃЏрЃЮрЃўрЃЌрЃ«рЃЮрЃЋрЃА handshake-рЃА

**рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ:**
- HTTP
- SSH
- рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃў

### UDP = рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃџрЃў рЃдрЃўрЃљ рЃЉрЃљрЃарЃљрЃЌрЃў

- Fire-and-forget
- рЃњрЃљрЃарЃљрЃюрЃбрЃўрЃљ рЃљрЃа рЃљрЃарЃўрЃА
- Handshake рЃљрЃа рЃљрЃарЃўрЃА
- рЃАрЃгрЃарЃљрЃцрЃў
- рЃЏрЃАрЃБрЃЉрЃБрЃЦрЃў

**рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ:**
- DNS рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў
- рЃЋрЃўрЃЊрЃћрЃЮ рЃАрЃбрЃарЃўрЃЏрЃўрЃюрЃњрЃў
- VoIP
- рЃЌрЃљрЃЏрЃљрЃерЃћрЃЉрЃў

## рЃюрЃљрЃгрЃўрЃџрЃў 6 Рђћ рЃАрЃЮрЃЎрЃћрЃбрЃўрЃА рЃАрЃљрЃАрЃўрЃфрЃЮрЃфрЃ«рЃџрЃЮ рЃфрЃўрЃЎрЃџрЃў (рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў!)

рЃарЃЮрЃфрЃљ рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃБрЃАрЃЏрЃћрЃюрЃА, рЃЏрЃљрЃА рЃљрЃЦрЃЋрЃА **LISTENING** рЃАрЃЮрЃЎрЃћрЃбрЃў.

рЃарЃЮрЃфрЃљ рЃЎрЃџрЃўрЃћрЃюрЃбрЃў рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ:
- рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ рЃљрЃ«рЃљрЃџрЃў **ESTABLISHED** рЃАрЃЮрЃЎрЃћрЃбрЃў рЃљрЃЏ рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃАрЃЌрЃЋрЃўрЃА.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

Nginx:
```

```

рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃўрЃљрЃю:
```
ESTAB   local-ip:443 Рєњ client-ip:51512
ESTAB   local-ip:443 Рєњ client-ip:51513
ESTAB   local-ip:443 Рєњ client-ip:51514
```

**рЃћрЃарЃЌ listening рЃАрЃЮрЃЎрЃћрЃбрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЏрЃўрЃўрЃдрЃЮрЃА рЃЏрЃўрЃџрЃўрЃЮрЃюрЃЮрЃЉрЃўрЃЌ established рЃАрЃЮрЃЎрЃћрЃбрЃў.**

## рЃюрЃљрЃгрЃўрЃџрЃў 8 Рђћ рЃ«рЃерЃўрЃарЃў рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃћрЃЉрЃў (рЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў)

рЃ«рЃЋрЃљрЃџ рЃдрЃарЃЏрЃљрЃЊ рЃЕрЃљрЃЋрЃБрЃдрЃарЃЏрЃљрЃЋрЃЊрЃћрЃЉрЃўрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљрЃА.
рЃЊрЃдрЃћрЃА, рЃАрЃгрЃарЃљрЃцрЃў рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ:

### ­Ъћ╣ LISTEN
рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃћрЃџрЃЮрЃЊрЃћрЃЉрЃљ рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃА.

### ­Ъћ╣ SYN-SENT / SYN-RECV
Handshake рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћрЃЮрЃЉрЃА.

### ­Ъћ╣ ESTABLISHED
рЃљрЃЦрЃбрЃўрЃБрЃарЃў рЃЎрЃљрЃЋрЃерЃўрЃарЃў.

### ­Ъћ╣ TIME_WAIT
рЃАрЃЮрЃЎрЃћрЃбрЃў рЃћрЃџрЃЮрЃЊрЃћрЃЉрЃљ рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљрЃЏрЃЊрЃћ.
(рЃ«рЃћрЃџрЃА рЃБрЃерЃџрЃўрЃА рЃБрЃфрЃюрЃљрЃБрЃа рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃарЃћрЃЋрЃљрЃА.)

### ­Ъћ╣ CLOSE_WAIT
рЃЊрЃўрЃАрЃбрЃљрЃюрЃфрЃўрЃБрЃарЃЏрЃљ рЃЊрЃљрЃ«рЃБрЃарЃљ, рЃЌрЃЦрЃЋрЃћрЃюрЃЏрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃЏ рЃљрЃарЃљ.

### ­Ъћ╣ FIN_WAIT
рЃЊрЃљрЃ«рЃБрЃарЃЋрЃўрЃА handshake рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћрЃЮрЃЉрЃА.

рЃћрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃћрЃЉрЃў рЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃарЃЮрЃфрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ:
```bash
ss -tan
```

**рЃ«рЃЋрЃљрЃџ рЃдрЃарЃЏрЃљрЃЊ рЃљрЃЋрЃ«рЃАрЃюрЃўрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃљрЃЌрЃњрЃљрЃюрЃА.**

## рЃюрЃљрЃгрЃўрЃџрЃў 9 Рђћ рЃарЃљ рЃ«рЃЊрЃћрЃЉрЃљ рЃарЃЮрЃфрЃљ рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃЌ:

1№ИЈРЃБ DNS рЃљрЃЏрЃЮрЃ«рЃАрЃюрЃўрЃА `google.com` Рєњ IP
2№ИЈРЃБ рЃЌрЃЦрЃЋрЃћрЃюрЃў kernel рЃўрЃарЃЕрЃћрЃЋрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ рЃгрЃДрЃљрЃарЃЮрЃА рЃърЃЮрЃарЃбрЃА - рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ: `55012`
3№ИЈРЃБ Kernel рЃЦрЃЏрЃюрЃўрЃА рЃАрЃЮрЃЎрЃћрЃбрЃА:
```
10.0.0.5:55012 Рєњ 142.250.183.14:443
```
4№ИЈРЃБ TCP handshake рЃўрЃгрЃДрЃћрЃЉрЃљ
5№ИЈРЃБ TLS handshake
6№ИЈРЃБ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃфрЃћрЃЏрЃљ
7№ИЈРЃБ рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃўрЃ«рЃБрЃарЃћрЃЉрЃљ (FIN, TIME_WAIT)

**рЃ«рЃЋрЃљрЃџ рЃЊрЃљрЃЋрЃерЃџрЃўрЃЌ handshake-рЃА.**

## рЃюрЃљрЃгрЃўрЃџрЃў 10 Рђћ рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ: рЃарЃЮрЃњрЃЮрЃа рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃў Multi-Tasking-рЃА

рЃгрЃљрЃарЃЏрЃЮрЃўрЃЊрЃњрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ рЃЌрЃЦрЃЋрЃћрЃю рЃњрЃДрЃљрЃЋрЃЌ 10 рЃЏрЃћрЃњрЃЮрЃЉрЃљрЃарЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃАрЃбрЃБрЃЏрЃарЃЮрЃЉрЃћрЃю рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃАрЃљрЃ«рЃџрЃА.
рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЎрЃљрЃЎрЃБрЃюрЃћрЃЉрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЎрЃљрЃарЃќрЃћ (рЃърЃЮрЃарЃбрЃў).

рЃЌрЃЦрЃЋрЃћрЃю:
- рЃ«рЃАрЃюрЃўрЃЌ рЃЎрЃљрЃарЃА 22 Рєњ рЃАрЃљрЃБрЃЉрЃарЃЮрЃЉрЃЌ SSH-рЃќрЃћ
- рЃ«рЃАрЃюрЃўрЃЌ рЃЎрЃљрЃарЃА 80 Рєњ рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃЌ рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃА
- рЃ«рЃАрЃюрЃўрЃЌ рЃЎрЃљрЃарЃА 3306 Рєњ рЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃўрЃЌ MySQL-рЃА
- рЃ«рЃАрЃюрЃўрЃЌ рЃЎрЃљрЃарЃА 5432 Рєњ рЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃўрЃЌ Postgres-рЃА

**рЃДрЃЋрЃћрЃџрЃљ рЃЎрЃљрЃарЃў = рЃфрЃљрЃџрЃЎрЃћ рЃАрЃљрЃБрЃЉрЃљрЃарЃў.**

рЃАрЃгрЃЮрЃарЃћрЃЊ рЃћрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃърЃЮрЃарЃбрЃћрЃЉрЃў.

## рЃюрЃљрЃгрЃўрЃџрЃў 11 Рђћ рЃарЃћрЃљрЃџрЃБрЃарЃў DevOps/SRE рЃЊрЃћрЃЉрЃљрЃњрЃўрЃюрЃњрЃўрЃА рЃАрЃфрЃћрЃюрЃљрЃарЃћрЃЉрЃў

### рЃАрЃфрЃћрЃюрЃљрЃарЃў 1 Рђћ "рЃърЃЮрЃарЃбрЃў рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ"

```bash
sudo lsof -i :8080
```

**рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋ:**
- рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃюрЃАрЃў рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃўрЃљ
- рЃќрЃЮрЃЏрЃЉрЃў рЃърЃарЃЮрЃфрЃћрЃАрЃў
- рЃАрЃ«рЃЋрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃў рЃЏрЃўрЃЉрЃЏрЃБрЃџрЃўрЃљ рЃљрЃЏ рЃърЃЮрЃарЃбрЃќрЃћ

**рЃњрЃљрЃЏрЃЮрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ:**
```bash
sudo fuser -k 8080/tcp
```

### рЃАрЃфрЃћрЃюрЃљрЃарЃў 2 Рђћ "рЃЕрЃћрЃЏ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃА рЃљрЃа рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ рЃърЃЮрЃарЃбрЃќрЃћ <1024> root-рЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ"

рЃърЃЮрЃарЃбрЃћрЃЉрЃў 0РђЊ1023 рЃљрЃарЃўрЃА рЃърЃарЃўрЃЋрЃўрЃџрЃћрЃњрЃўрЃарЃћрЃЉрЃБрЃџрЃў.

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃћрЃЉрЃў:**
- рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃарЃЮрЃњрЃЮрЃарЃф root (рЃљрЃа рЃљрЃарЃўрЃА рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў)
- рЃљрЃю рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ capabilities:
```bash
sudo setcap 'cap_net_bind_service=+ep' /usr/bin/myapp
```

### рЃАрЃфрЃћрЃюрЃљрЃарЃў 3 Рђћ "рЃФрЃљрЃџрЃўрЃљрЃю рЃЉрЃћрЃЋрЃарЃў TIME_WAIT рЃЎрЃљрЃЋрЃерЃўрЃарЃў"

**рЃ«рЃерЃўрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃгрЃЋрЃћрЃБрЃџрЃўрЃљ:**
- рЃбрЃЋрЃўрЃарЃЌрЃўрЃА рЃњрЃљрЃЏрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃџрЃћрЃЉрЃўрЃЌ
- рЃЏрЃЮрЃЎрЃџрЃћрЃЋрЃљрЃЊрЃўрЃљрЃюрЃў HTTP рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃўрЃЌ
- рЃЏрЃљрЃдрЃљрЃџрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃАрЃўрЃ«рЃерЃўрЃарЃўрЃЌ

**рЃњрЃљрЃЏрЃЮрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ:**
- рЃЕрЃљрЃарЃЌрЃћрЃЌ keep-alive
- рЃЊрЃљрЃљрЃарЃћрЃњрЃБрЃџрЃўрЃарЃћрЃЌ kernel рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў
- рЃљрЃю рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ connection pooling

### рЃАрЃфрЃћрЃюрЃљрЃарЃў 4 Рђћ "Kubernetes pod-рЃА рЃљрЃа рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ рЃърЃЮрЃарЃбрЃќрЃћ"

Pod security policy рЃљрЃю securityContext рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЉрЃџрЃЮрЃЎрЃљрЃЋрЃЊрЃћрЃА рЃърЃЮрЃарЃбрЃўрЃА рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљрЃА.

### рЃАрЃфрЃћрЃюрЃљрЃарЃў 5 Рђћ "рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃЏрЃўрЃЎрЃарЃЮрЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃерЃў рЃФрЃљрЃџрЃўрЃљрЃю рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃАрЃљрЃБрЃЉрЃарЃЮрЃЉрЃА"

Ephemeral рЃърЃЮрЃарЃбрЃћрЃЉрЃўрЃА рЃљрЃЏрЃЮрЃгрЃБрЃарЃЋрЃљ:
- рЃФрЃљрЃџрЃўрЃљрЃю рЃЉрЃћрЃЋрЃарЃў рЃњрЃљрЃЏрЃљрЃЋрЃљрЃџрЃў рЃЎрЃљрЃЋрЃерЃўрЃарЃў
- рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃБрЃДрЃћрЃюрЃћрЃЉрЃџрЃЮрЃЉрЃљ

**рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ:**
```bash
ss -s
```

**рЃњрЃљрЃЏрЃЮрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ:**
- рЃњрЃљрЃќрЃљрЃарЃЊрЃћрЃЌ ephemeral рЃърЃЮрЃарЃбрЃћрЃЉрЃўрЃА рЃЊрЃўрЃљрЃърЃљрЃќрЃЮрЃюрЃў
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ HTTP connection pooling

## рЃюрЃљрЃгрЃўрЃџрЃў 12 Рђћ рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў (рЃЊрЃдрЃћ 9)

### рЃДрЃЋрЃћрЃџрЃљ рЃАрЃЮрЃЎрЃћрЃбрЃўрЃА рЃАрЃўрЃљ (TCP + UDP)

## рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ (рЃЊрЃдрЃћ 9)

рЃЊрЃдрЃћрЃА рЃўрЃАрЃгрЃљрЃЋрЃџрЃћрЃЌ:
- РюЁ рЃарЃљ рЃљрЃарЃўрЃА рЃърЃЮрЃарЃбрЃћрЃЉрЃў (рЃЎрЃљрЃарЃћрЃЉрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА)
- РюЁ рЃарЃљ рЃљрЃарЃўрЃА рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃў (IP + рЃърЃЮрЃарЃбрЃў + рЃърЃарЃЮрЃбрЃЮрЃЎрЃЮрЃџрЃў)
- РюЁ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљ рЃАрЃћрЃарЃЋрЃћрЃа + рЃЎрЃџрЃўрЃћрЃюрЃб рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА
- РюЁ Ephemeral рЃърЃЮрЃарЃбрЃћрЃЉрЃў
- РюЁ TCP vs UDP рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃў
- РюЁ рЃАрЃЮрЃЎрЃћрЃбрЃўрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃћрЃЉрЃў (LISTEN, ESTAB, TIME_WAITРђд)
- РюЁ рЃарЃЮрЃњрЃЮрЃа рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА Linux-рЃў рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃА
- РюЁ рЃарЃЮрЃњрЃЮрЃа рЃЊрЃћрЃЉрЃљрЃњрЃЊрЃћрЃЉрЃљ рЃърЃЮрЃарЃбрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃў `ss`, `lsof`, `fuser`-рЃўрЃЌ
- РюЁ рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃърЃЮрЃарЃбрЃћрЃЉрЃў рЃњрЃљрЃЊрЃљрЃЏрЃгрЃДрЃЋрЃћрЃбрЃў рЃЏрЃўрЃЎрЃарЃЮрЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃерЃў, Kubernetes-рЃерЃў & cloud рЃЦрЃАрЃћрЃџрЃћрЃЉрЃерЃў

**рЃћрЃА рЃљрЃарЃўрЃА рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃАрЃљрЃЏрЃерЃћрЃюрЃћрЃЉрЃџрЃЮ рЃЉрЃџрЃЮрЃЎрЃў рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЦрЃАрЃћрЃџрЃћрЃЉрЃўрЃАрЃљ.**
