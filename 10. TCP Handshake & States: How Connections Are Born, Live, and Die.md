# рЃЊрЃдрЃћ 10 Рђћ TCP Handshake & States: рЃарЃЮрЃњрЃЮрЃа рЃўрЃЉрЃљрЃЊрЃћрЃЉрЃўрЃљрЃю, рЃфрЃ«рЃЮрЃЋрЃарЃЮрЃЉрЃћрЃю рЃЊрЃљ рЃЎрЃЋрЃЊрЃћрЃЉрЃўрЃљрЃю рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃў (рЃљрЃ«рЃАрЃюрЃўрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃўрЃАрЃбрЃЮрЃарЃўрЃљ)

рЃдрЃарЃЏрЃљ рЃЕрЃљрЃФрЃўрЃарЃЋрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃљрЃарЃЎрЃЋрЃћрЃЋрЃА SYN, SYN-ACK, FIN, TIME_WAIT, CLOSE_WAIT, RST рЃЊрЃљ рЃЏрЃћрЃбрЃА Рђћ рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃЌ, рЃюрЃљрЃЎрЃљрЃЊрЃўрЃА рЃЊрЃўрЃљрЃњрЃарЃљрЃЏрЃћрЃЉрЃўрЃЌ рЃЊрЃљ рЃарЃћрЃљрЃџрЃБрЃарЃў SRE рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃўрЃЌ.

## рЃарЃљрЃбрЃЮрЃЏ рЃўрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃА TCP-рЃЏ рЃЏрЃЌрЃћрЃџрЃў рЃЊрЃдрЃћ

TCP рЃљрЃарЃўрЃА рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃърЃарЃЮрЃбрЃЮрЃЎрЃЮрЃџрЃў, рЃарЃљрЃф рЃЮрЃЊрЃћрЃАрЃЏрЃћ рЃўрЃЦрЃюрЃљ рЃњрЃљрЃЏрЃЮрЃњрЃЮрЃюрЃћрЃЉрЃБрЃџрЃў.

рЃДрЃЮрЃЋрЃћрЃџ рЃ»рЃћрЃарЃќрЃћ, рЃарЃЮрЃфрЃљ рЃЌрЃЦрЃЋрЃћрЃю:
- рЃ«рЃАрЃюрЃўрЃЌ рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃА
- рЃБрЃФрЃљрЃ«рЃћрЃЉрЃЌ REST API-рЃА
- рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃўрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃА
- SSH-рЃўрЃЌ рЃерЃћрЃЊрЃўрЃ«рЃљрЃарЃЌ рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ
- рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃЌ POST-рЃА рЃЏрЃўрЃЎрЃарЃЮрЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ
- рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃЌ рЃџрЃЮрЃњрЃћрЃЉрЃА Elasticsearch-рЃерЃў
- рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ Kubernetes рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃА
- рЃАрЃљрЃБрЃЉрЃарЃЮрЃЉрЃЌ рЃбрЃЋрЃўрЃарЃЌрЃўрЃА рЃњрЃљрЃЏрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃћрЃџрЃЌрЃљрЃю

**рЃЌрЃЦрЃЋрЃћрЃю рЃћрЃДрЃарЃЊрЃюрЃЮрЃЉрЃўрЃЌ TCP-рЃА.**

рЃЏрЃљрЃњрЃарЃљрЃЏ рЃБрЃЏрЃћрЃбрЃћрЃАрЃў рЃўрЃюрЃЪрЃўрЃюрЃћрЃарЃў рЃљрЃарЃљрЃАрЃЮрЃЊрЃћрЃА рЃћрЃАрЃЏрЃўрЃА рЃАрЃарЃБрЃџрЃљрЃЊ:
- рЃарЃЮрЃњрЃЮрЃа рЃўрЃгрЃДрЃћрЃЉрЃљ TCP рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃў
- рЃарЃЮрЃњрЃЮрЃа рЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃўрЃАрЃўрЃюрЃў
- рЃарЃљрЃбрЃЮрЃЏ рЃўрЃГрЃћрЃарЃА рЃўрЃАрЃўрЃюрЃў
- рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА TIME_WAIT
- рЃарЃљрЃбрЃЮрЃЏ рЃ«рЃЊрЃћрЃЉрЃљ CLOSE_WAIT
- рЃарЃЮрЃњрЃЮрЃа рЃћрЃарЃћрЃЋрЃўрЃљрЃю рЃцрЃљрЃўрЃарЃЋрЃЮрЃџрЃћрЃЉрЃў
- рЃарЃљ рЃљрЃарЃўрЃА SYN flood-рЃћрЃЉрЃў

рЃЊрЃдрЃћрЃА рЃњрЃљрЃЦрЃЋрЃЌ рЃАрЃарЃБрЃџрЃў рЃАрЃўрЃфрЃ«рЃљрЃЊрЃћ Рђћ рЃўрЃАрЃбрЃЮрЃарЃўрЃўрЃАрЃћрЃЉрЃарЃў рЃљрЃ«рЃАрЃюрЃўрЃЌ.

рЃЊрЃљрЃЋрЃўрЃгрЃДрЃЮрЃЌ.

## рЃюрЃљрЃгрЃўрЃџрЃў 1 Рђћ TCP рЃљрЃарЃўрЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃбрЃћрЃџрЃћрЃцрЃЮрЃюрЃўрЃА рЃќрЃљрЃарЃў

рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃў рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃўрЃА рЃљрЃАрЃљрЃ«рЃАрЃюрЃћрЃџрЃљрЃЊ.

рЃгрЃљрЃарЃЏрЃЮрЃўрЃЊрЃњрЃўрЃюрЃћрЃЌ рЃЮрЃарЃў рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃў: рЃћрЃџрЃўрЃАрЃў рЃЊрЃљ рЃЉрЃЮрЃЉрЃў.

- **рЃћрЃџрЃўрЃАрЃў = рЃЎрЃџрЃўрЃћрЃюрЃбрЃў**
- **рЃЉрЃЮрЃЉрЃў = рЃАрЃћрЃарЃЋрЃћрЃарЃў**

рЃарЃЮрЃфрЃљ рЃћрЃџрЃўрЃАрЃА рЃАрЃБрЃарЃА рЃћрЃАрЃљрЃБрЃЉрЃарЃЮрЃА рЃЉрЃЮрЃЉрЃА, рЃЏрЃљрЃю рЃБрЃюрЃЊрЃљ:
1. рЃЊрЃљрЃарЃћрЃЎрЃЮрЃА рЃЉрЃЮрЃЉрЃА
2. рЃЊрЃљрЃћрЃџрЃЮрЃЊрЃЮрЃА рЃарЃЮрЃЏ рЃЉрЃЮрЃЉрЃў рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃЮрЃА
3. рЃњрЃљрЃўрЃњрЃЮрЃА рЃарЃЮрЃЏ рЃЉрЃЮрЃЉрЃЏрЃљ рЃЌрЃЦрЃЋрЃљ рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ
4. рЃЏрЃљрЃерЃўрЃю рЃАрЃљрЃБрЃЉрЃљрЃарЃў рЃўрЃгрЃДрЃћрЃЉрЃљ

**рЃћрЃА рЃљрЃарЃўрЃА TCP 3-рЃњрЃќрЃўрЃљрЃюрЃў handshake.**

рЃљрЃ«рЃџрЃљ рЃЏрЃЮрЃЊрЃўрЃЌ рЃерЃћрЃЋрЃљрЃЊрЃљрЃарЃЮрЃЌ рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ:

![alt text](photo2/image-16.png)

## рЃюрЃљрЃгрЃўрЃџрЃў 2 Рђћ TCP 3-рЃњрЃќрЃўрЃљрЃюрЃў Handshake (рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃЊрЃљрЃЉрЃљрЃЊрЃћрЃЉрЃљ)

рЃЏрЃЮрЃЊрЃўрЃЌ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃљрЃЊ рЃЋрЃюрЃљрЃ«рЃЮрЃЌ.

### рЃюрЃљрЃЉрЃўрЃ»рЃў 1 Рђћ SYN (рЃЎрЃџрЃўрЃћрЃюрЃбрЃў Рєњ рЃАрЃћрЃарЃЋрЃћрЃарЃў)

рЃЎрЃџрЃўрЃћрЃюрЃбрЃў рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА SYN-рЃА:
```
SYN = рЃЏрЃўрЃюрЃЊрЃљ рЃЋрЃўрЃАрЃљрЃБрЃЉрЃарЃЮ. рЃ«рЃљрЃа рЃљрЃЦ?
Sequence number = X
```

### рЃюрЃљрЃЉрЃўрЃ»рЃў 2 Рђћ SYN-ACK (рЃАрЃћрЃарЃЋрЃћрЃарЃў Рєњ рЃЎрЃџрЃўрЃћрЃюрЃбрЃў)

рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃърЃљрЃАрЃБрЃ«рЃЮрЃЉрЃА:
```
SYN-ACK = рЃЊрЃўрЃљрЃ«, рЃљрЃЦ рЃЋрЃљрЃа. рЃЏрЃЮрЃЊрЃў рЃЋрЃўрЃАрЃљрЃБрЃЉрЃарЃЮрЃЌ.
Acknowledgment = X+1
рЃЕрЃћрЃЏрЃў sequence number = Y
```

### рЃюрЃљрЃЉрЃўрЃ»рЃў 3 Рђћ ACK (рЃЎрЃџрЃўрЃћрЃюрЃбрЃў Рєњ рЃАрЃћрЃарЃЋрЃћрЃарЃў)

рЃЎрЃџрЃўрЃћрЃюрЃбрЃў рЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃА:
```
ACK = рЃЎрЃљрЃарЃњрЃў, рЃЊрЃљрЃЋрЃўрЃгрЃДрЃЮрЃЌ.
Acknowledgment = Y+1
```

**рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃљрЃ«рЃџрЃљ рЃљрЃарЃўрЃА: ESTABLISHED**

### ASCII рЃЊрЃўрЃљрЃњрЃарЃљрЃЏрЃљ Рђћ TCP Handshake

```
рЃЎрЃџрЃўрЃћрЃюрЃбрЃў                                     рЃАрЃћрЃарЃЋрЃћрЃарЃў
  |                                            |
  |------------ SYN (seq = X) ---------------->|
  |                                            |
  |<------- SYN-ACK (seq = Y, ack = X+1) ------|
  |                                            |
  |------------ ACK (ack = Y+1) -------------->|
  |                                            |
рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃЊрЃљрЃЏрЃДрЃљрЃарЃћрЃЉрЃБрЃџрЃўрЃљ ------------------------+
```

## рЃюрЃљрЃгрЃўрЃџрЃў 3 Рђћ 4-рЃњрЃќрЃўрЃљрЃюрЃў Teardown (рЃарЃЮрЃњрЃЮрЃа рЃЎрЃЋрЃЊрЃћрЃЉрЃљ TCP рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃў)

TCP teardown рЃљрЃа рЃљрЃарЃўрЃА 3-рЃњрЃќрЃўрЃљрЃюрЃў.
рЃћрЃА рЃљрЃарЃўрЃА 4 рЃюрЃљрЃЉрЃўрЃ»рЃў.

**рЃарЃљрЃбрЃЮрЃЏ?**
рЃарЃљрЃЊрЃњрЃљрЃю рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃ«рЃљрЃарЃћ рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃџрЃљрЃЊ рЃгрЃДрЃЋрЃћрЃбрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљрЃА.

### рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ:

рЃбрЃћрЃџрЃћрЃцрЃЮрЃюрЃўрЃА рЃќрЃљрЃарЃўрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃљ:
1. рЃЌрЃЦрЃЋрЃћрЃю рЃљрЃЏрЃЉрЃЮрЃЉрЃЌ "рЃюрЃљрЃ«рЃЋрЃљрЃЏрЃЊрЃўрЃА, рЃЏрЃћ рЃЊрЃљрЃЋрЃљрЃАрЃарЃБрЃџрЃћ рЃАрЃљрЃБрЃЉрЃљрЃарЃў."
2. рЃЏрЃћрЃЮрЃарЃћ рЃърЃўрЃарЃў рЃљрЃЏрЃЉрЃЮрЃЉрЃА "рЃЎрЃљрЃарЃњрЃў, рЃЏрЃћ рЃЎрЃўрЃЊрЃћрЃЋ рЃфрЃЮрЃбрЃљ рЃЋрЃўрЃАрЃљрЃБрЃЉрЃарЃћрЃЉ."
3. рЃўрЃАрЃўрЃюрЃў рЃљрЃЏрЃЌрЃљрЃЋрЃарЃћрЃЉрЃћрЃю рЃАрЃљрЃБрЃЉрЃљрЃарЃА.
4. рЃўрЃАрЃўрЃюрЃў рЃљрЃЏрЃЉрЃЮрЃЉрЃћрЃю "рЃљрЃ«рЃџрЃљ рЃЏрЃћрЃф рЃЊрЃљрЃЋрЃљрЃАрЃарЃБрЃџрЃћ."
5. рЃќрЃљрЃарЃў рЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ.

рЃЏрЃЮрЃЊрЃўрЃЌ рЃерЃћрЃЋрЃљрЃЊрЃљрЃарЃЮрЃЌ TCP-рЃА.

### рЃюрЃљрЃЉрЃўрЃ»рЃў 1 Рђћ FIN рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃАрЃњрЃљрЃю

```
рЃЏрЃћ рЃЊрЃљрЃЋрЃљрЃАрЃарЃБрЃџрЃћ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ.
```
рЃЎрЃџрЃўрЃћрЃюрЃбрЃў рЃерЃћрЃЊрЃўрЃА: **FIN_WAIT_1**

### рЃюрЃљрЃЉрЃўрЃ»рЃў 2 Рђћ ACK рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃАрЃњрЃљрЃю

```
рЃЏрЃўрЃЋрЃўрЃдрЃћ рЃерЃћрЃюрЃў FIN.
```
рЃЎрЃџрЃўрЃћрЃюрЃбрЃў рЃерЃћрЃЊрЃўрЃА: **FIN_WAIT_2**
рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃерЃћрЃЊрЃўрЃА: **CLOSE_WAIT**

### рЃюрЃљрЃЉрЃўрЃ»рЃў 3 Рђћ FIN рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃАрЃњрЃљрЃю

```
рЃЏрЃћрЃф рЃЊрЃљрЃЋрЃљрЃАрЃарЃБрЃџрЃћ рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ.
```

### рЃюрЃљрЃЉрЃўрЃ»рЃў 4 Рђћ ACK рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃАрЃњрЃљрЃю

```
рЃЏрЃўрЃЋрЃўрЃдрЃћ рЃерЃћрЃюрЃў FIN.
```

## рЃюрЃљрЃгрЃўрЃџрЃў 4 Рђћ рЃарЃљрЃбрЃЮрЃЏ рЃ«рЃЊрЃћрЃЉрЃљ TIME_WAIT (рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃљрЃ«рЃАрЃюрЃўрЃџрЃў)

**TIME_WAIT рЃљрЃа рЃљрЃарЃўрЃА рЃЉрЃљрЃњрЃў.**
**рЃћрЃА рЃљрЃарЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў.**

TIME_WAIT = "рЃЏрЃЮрЃЏрЃћ рЃфрЃЮрЃбрЃљ рЃЊрЃљрЃЋрЃџрЃЮрЃЊрЃЮ рЃАрЃарЃБрЃџрЃўрЃљрЃЊ рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљрЃЏрЃЊрЃћ рЃўрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃЌрЃБ рЃФрЃЋрЃћрЃџрЃў рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў рЃ»рЃћрЃа рЃЎрЃўрЃЊрЃћрЃЋ рЃЊрЃљрЃ«рЃћрЃбрЃўрЃљрЃџрЃЮрЃЉрЃћрЃю рЃЦрЃАрЃћрЃџрЃерЃў."

### рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА TIME_WAIT:

**TIME_WAIT р▓Др▓Юр▓Ћр▓ћр▓џр▓Ќр▓Ћр▓ўр▓А рЃ«рЃЊрЃћрЃЉрЃљ рЃўрЃЏ рЃЏрЃ«рЃљрЃарЃћрЃќрЃћ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ ACK-рЃА.**

## рЃюрЃљрЃгрЃўрЃџрЃў 5 Рђћ рЃарЃљрЃбрЃЮрЃЏ рЃ«рЃЊрЃћрЃЉрЃљ CLOSE_WAIT (рЃЊрЃљ рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃАрЃљрЃерЃўрЃерЃў)

- рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЪрЃЮрЃюрЃЋрЃљ

**рЃњрЃљрЃЏрЃЮрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ:**
- рЃЕрЃљрЃарЃЌрЃћрЃЌ HTTP keep-alive
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ connection pooling
- рЃЊрЃљрЃљрЃарЃћрЃњрЃБрЃџрЃўрЃарЃћрЃЌ TIME_WAIT рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў (рЃцрЃарЃЌрЃ«рЃўрЃџрЃљрЃЊ)

### рЃАрЃфрЃћрЃюрЃљрЃарЃў 3 Рђћ "рЃЕрЃћрЃЏрЃў рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃБрЃџрЃўрЃљ SYN_RECV-рЃўрЃЌ"

**рЃЏрЃўрЃќрЃћрЃќрЃў:**
- SYN flood
- рЃцрЃљрЃўрЃарЃЋрЃЮрЃџрЃў/reverse proxy рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃў

**рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ:**
```bash
ss -tan | grep SYN_RECV
```

**рЃњрЃљрЃЏрЃЮрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ:**
рЃЕрЃљрЃарЃЌрЃћрЃЌ SYN cookies:
```bash
sysctl -w net.ipv4.tcp_syncookies=1
```

### рЃАрЃфрЃћрЃюрЃљрЃарЃў 4 Рђћ "рЃЎрЃџрЃўрЃћрЃюрЃбрЃў рЃБрЃАрЃљрЃАрЃарЃБрЃџрЃЮрЃЊ рЃфрЃЊрЃўрЃџрЃЮрЃЉрЃА рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃЎрЃљрЃарЃњрЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА"

**рЃАрЃљрЃЋрЃљрЃарЃљрЃБрЃЊрЃЮ рЃЏрЃўрЃќрЃћрЃќрЃў:**
- рЃцрЃљрЃўрЃарЃЋрЃЮрЃџрЃў рЃЉрЃџрЃЮрЃЎрЃљрЃЋрЃА SYN-ACK-рЃА
- рЃфрЃБрЃЊрЃў security group
- рЃЏрЃљрЃарЃерЃарЃБрЃбрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃљрЃАрЃўрЃЏрЃћрЃбрЃарЃўрЃљ

**рЃЊрЃћрЃЉрЃљрЃњрЃў:**
```bash
tcpdump -i eth0 'tcp[tcpflags] & tcp-syn != 0'
```

### рЃАрЃфрЃћрЃюрЃљрЃарЃў 5 Рђћ "рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃў рЃўрЃГрЃћрЃарЃћрЃю LAST_ACK-рЃерЃў"

**рЃЏрЃўрЃќрЃћрЃќрЃў:**
- рЃЎрЃџрЃўрЃћрЃюрЃбрЃў рЃљрЃа рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА close handshake-рЃА
- рЃЦрЃАрЃћрЃџрЃўрЃА drop FIN-рЃўрЃА рЃЊрЃарЃЮрЃА
- рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ

### рЃАрЃфрЃћрЃюрЃљрЃарЃў 6 Рђћ "рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃў рЃљрЃарЃљрЃАрЃЮрЃЊрЃћрЃА рЃЌрЃљрЃЋрЃўрЃАрЃБрЃцрЃџрЃЊрЃћрЃЉрЃљ"

**рЃЏрЃўрЃќрЃћрЃќрЃў:**
- рЃЎрЃџрЃўрЃћрЃюрЃб рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃљрЃа рЃ«рЃБрЃарЃљрЃЋрЃА рЃАрЃЮрЃЎрЃћрЃбрЃћрЃЉрЃА
- рЃљрЃю рЃАрЃћрЃарЃЋрЃћрЃарЃў рЃљрЃа рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА FIN-рЃА

## рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ (рЃЊрЃдрЃћ 10)

рЃЊрЃдрЃћрЃА рЃўрЃАрЃгрЃљрЃЋрЃџрЃћрЃЌ:
- РюЁ TCP-рЃўрЃА 3-рЃњрЃќрЃўрЃљрЃюрЃў handshake
- РюЁ 4-рЃњрЃќрЃўрЃљрЃюрЃў teardown
- РюЁ рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА TIME_WAIT
- РюЁ рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА CLOSE_WAIT рЃАрЃљрЃерЃўрЃерЃў
- РюЁ TCP state machine
- РюЁ SYN floods & SYN cookies
- РюЁ рЃарЃЮрЃњрЃЮрЃа рЃљрЃарЃдрЃЋрЃћрЃЋрЃћрЃю рЃцрЃљрЃўрЃарЃЋрЃЮрЃџрЃћрЃЉрЃў handshakes-рЃА
- РюЁ SRE рЃЊрЃћрЃЉрЃљрЃњрЃў рЃњрЃљрЃГрЃћрЃЊрЃўрЃџрЃў рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃўрЃАрЃљ
- РюЁ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ `tcpdump`, `ss` рЃЊрЃљ `netstat`

**TCP рЃљрЃарЃўрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃб рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃў.**
**рЃЏрЃўрЃАрЃў рЃњрЃљрЃњрЃћрЃЉрЃљ рЃњрЃ«рЃЊрЃўрЃЌ рЃБрЃЎрЃћрЃЌрЃћрЃА DevOps-рЃА, SRE-рЃА, backend рЃўрЃюрЃЪрЃўрЃюрЃћрЃарЃА рЃљрЃю cloud рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃЮрЃарЃА.**

## рЃЏрЃЮрЃЏрЃЊрЃћрЃЋрЃюрЃЮ (рЃЊрЃдрЃћ 11)

**UDP vs TCP: рЃарЃћрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃћрЃЉрЃў рЃљрЃ«рЃАрЃюрЃўрЃџрЃў (DevOps рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃўрЃЌ)**

рЃљрЃЋрЃ«рЃАрЃюрЃўрЃЌ:
- ­Ъћ╣ рЃАрЃљрЃюрЃЊрЃЮрЃЮрЃЉрЃљ
- ­Ъћ╣ рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃљ
- ­Ъћ╣ рЃАрЃўрЃЕрЃЦрЃљрЃарЃћ
- ­Ъћ╣ рЃќрЃћрЃЊрЃюрЃљрЃЊрЃћрЃЉрЃў
- ­Ъћ╣ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃў (DNS, VoIP, streaming, QUICРђд)
- ­Ъћ╣ рЃарЃљрЃбрЃЮрЃЏ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА Kubernetes UDP-рЃА DNS-рЃўрЃАрЃЌрЃЋрЃўрЃА
- ­Ъћ╣ UDP multiplexing рЃбрЃЋрЃўрЃарЃЌрЃўрЃА рЃњрЃљрЃЏрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃџрЃћрЃЉрЃерЃў
- ­Ъћ╣ рЃарЃЮрЃЊрЃўрЃА рЃљрЃўрЃарЃЕрЃўрЃЮрЃЌ TCP vs UDP
- ­Ъћ╣ рЃърЃљрЃЎрЃћрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃћрЃЉрЃў
- ­Ъћ╣ UDP рЃбрЃарЃљрЃцрЃўрЃЎрЃўрЃА рЃЊрЃћрЃЉрЃљрЃњрЃў

**рЃћрЃА рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЎрЃўрЃЊрЃћрЃЋ рЃћрЃарЃЌрЃў рЃцрЃљрЃюрЃбрЃљрЃАрЃбрЃўрЃЎрЃБрЃарЃў рЃЌрЃљрЃЋрЃў.**
